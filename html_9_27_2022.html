Headings

Headings communicate the organization of the content on the page page. Web browsers, plug-ins, and assistive. technologies can use them to provide in-page navigation (Reference)

Headings are represented by: < h1 > , < h2 >, < h3 >, < h4 >, < h5 >, < h6 >

As far as I can see, headings are vital to the website. Without them finding the information would be very difficult

In this website it is possible to find interesting information about headings, for example, about nesting

Header

The header element represents a container for introductory content or a set of navigational links. (Reference)

Headers are represented by: < header >

A header element typically contains:

  • One or more heading elements (< h1 > - < h6 >)
  • logo or icon
  • autorship information

Headers have the function of: a) help navigation throughtout the website, b) show the credentials of the company, c) give the title of the website

Footer

The footer, represented as: < footer > HTML element represents a footer for its nearest ancestor sectioning content or sectioning root element. A footer typically contains information about the author of the section, copyright data or links to related documents.

A <footer> usually contains:

  • authorship information
  • copyright information
  • contact information
  • sitemap
  • back to top links
  • related documents

A large number of websites have a footer. It seems not very important until it is used to put inportant information such as address or phone number. I have used the information contained in footer many times, for example when my grandfather needed a doctor.
